# 🧠 NLP Using Python

A comprehensive introduction to **Natural Language Processing (NLP)** using Python, covering core text processing techniques through hands-on Jupyter Notebooks.

---

## 📖 About

This repository is designed for beginners and intermediate learners who want to understand the fundamentals of NLP. It walks through essential text preprocessing and linguistic analysis techniques using popular Python libraries.

---

## 📂 Repository Structure

```
NLP-using-python/
├── NLP_USing_Python.ipynb       # Core NLP concepts and introduction
├── NLP_Using_Python_1.ipynb     # Extended NLP techniques - Part 1
├── NLP_Using_Python_2.ipynb     # Extended NLP techniques - Part 2
└── README.md
```

---

## ✨ Topics Covered

- **Tokenization** — Splitting text into words or sentences
- **Lemmatization** — Reducing words to their base/root form (e.g., "running" → "run")
- **Stemming** — Cutting words to their stem (e.g., "studies" → "studi")
- **POS Tagging** — Labeling parts of speech (noun, verb, adjective, etc.)
- **Text Corpora** — Working with structured collections of text data
- **And more fundamental NLP concepts**

## 🚀 Getting Started

### Prerequisites

```bash
pip install nltk jupyter
```

### Download NLTK Data

```python
import nltk
nltk.download('all')
```

### Run the Notebooks

```bash
git clone https://github.com/nauman07/NLP-using-python.git
cd NLP-using-python
jupyter notebook
```

Open any of the `.ipynb` files in your browser to get started.
